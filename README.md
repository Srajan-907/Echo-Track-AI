# EcoTrack AI - Advanced Carbon Footprint Tracker

EcoTrack AI is an advanced, privacy-first progressive web sandbox application designed to help users track, analyze, and optimize their daily carbon footprint using dynamic profiling algorithms and smart habit building. 

The app features a responsive mobile-first UI wrapper, real-time analytics graphs, a dynamic target budgeting system, and an immediate client-side theme switcher.

## Key Features

* **AI-Calibrated Carbon Budgeting:** Sets personalized daily carbon thresholds based on transport metrics, diet configurations, and aggressive/steady reduction strategies.
* **Dynamic Light & Dark Mode:** A full theme-switching engine that instantly adapts backgrounds, cards, inputs, and text readability.
* **Live Chart.js Micro-Analytics:** Visualizes weekly emission trends. Grid lines and typography automatically re-render dynamically depending on whether Light or Dark mode is selected.
* **Activity Streams Log:** Real-time chronological activity feed with running calculation adapters that log transport, energy, meals, and retail metrics instantly.
* **Interactive Habit Optimizers:** Gamified challenge task cards that deduct carbon offsets from your daily balance and reward user point profiles.
* **Smart Notification System:** Slide-in feedback alert toasts confirm metrics data changes without interrupting app navigation workflows.

## Architecture & Tech Stack

* **Front-end Structure:** HTML5 (Semantic Structure)
* **Styling Framework:** Tailwind CSS CDN v3 (with custom `class` Dark Mode configuration layer)
* **Icons Package:** FontAwesome v6
* **Data Visualization:** Chart.js
